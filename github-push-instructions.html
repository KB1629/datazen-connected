GitHub Push Instructions for DataZen

Follow these instructions to push your code to GitHub repository: https://github.com/KB1629/DataZen.git

Method 1: Using Command Line

Open a new PowerShell window and run the following commands:

cd C:\nifi-api-integration\DataZen # Add all files git add . # Commit changes git commit -m "Add NiFi integration with certificate-based authentication" # Configure remote with your username git remote set-url origin https://KB1629@github.com/KB1629/DataZen.git # Push to GitHub (you'll be prompted for password/token) git push origin main

Note: If you have two-factor authentication enabled on GitHub, you must use a personal access token instead of your password.

Method 2: Using a Personal Access Token

If you have two-factor authentication enabled or regular password authentication doesn't work:

  1. Go to https://github.com/settings/tokens
  2. Click "Generate new token" (classic)
  3. Give it a name like "DataZen Push"
  4. Select the "repo" scope
  5. Click "Generate token" and copy the token

Then use the token in your command line:

cd C:\nifi-api-integration\DataZen # Set the remote URL with token git remote set-url origin https://KB1629:YOUR_TOKEN@github.com/KB1629/DataZen.git # Push to GitHub git push origin main # Reset the remote URL for security git remote set-url origin https://github.com/KB1629/DataZen.git

Replace YOUR_TOKEN with the actual token you generated.

Method 3: Using GitHub Desktop

If command line doesn't work for you:

  1. Install GitHub Desktop
  2. Open GitHub Desktop and login to your account
  3. Add your local repository (File > Add local repository)
  4. Navigate to C:\nifi-api-integration\DataZen
  5. Commit your changes and push to GitHub

Method 4: Manual Upload

If all else fails, you can manually upload your files:

  1. Go to https://github.com/KB1629/DataZen
  2. Click "Add file" > "Upload files"
  3. Drag and drop your files or select them using the file browser
  4. Add a commit message and click "Commit changes"

You may need to do this multiple times for different folders if there are many files.
